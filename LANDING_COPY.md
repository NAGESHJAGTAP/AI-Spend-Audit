## FAQ Section (5 Real Q&As)

### Q1: Is this tool free to use?
**A:** Yes, the tool is completely free. We do not require a credit card or registration to calculate your savings. We show you the full results first. For high-savings teams, we offer optional expert consultation services to implement changes.

### Q2: How does the savings engine work?
**A:** The engine evaluates your current subscriptions using a hardcoded, deterministic rules database updated weekly. It flags redundancies (like paying for both Cursor and GitHub Copilot for the same developers), seat count minimum violations (such as paying for Claude Team with fewer than 5 members), and suggests credit pools.

### Q3: What happens to my data?
**A:** We take privacy seriously. All identifying details (name, company, role, email) are kept strictly confidential and stored in a secure database. Public share links are anonymized and strip all names and emails—only showing the tool names and calculation numbers.

### Q4: What is "API Direct" and why should I include it?
**A:** Many teams pay for retail web interfaces (like ChatGPT Plus) when their usage would be cheaper through direct API keys, or they route simple tasks to premium API models (like GPT-4o) when lightweight models (like GPT-4o-mini) would save 90% of the cost. The audit engine calculates these optimizations.

### Q5: Who is Techvruk?
**A:** Techvruk is an AI operations and infrastructure consultancy. We build and deploy AI tools for startups. We built this tool to help early-stage teams streamline their infrastructure costs so they can focus their budgets on shipping core products.
